Bem-vindo ao meu site de exemplo flexbox

Este site utiliza um arquivo CSS separando, para criar o menu horizontal com flexbox, além de uma aparaencia limpa e moderna

As tags exigidas estão todas presente: <head>, <main>, <div>, (container interno) <hi> entre outros.

O meu horizonte é abtido com display: flexna lista <ul>. O CSS externo facilita a manutenção e o aprendizado separado de estrutura e da apresentação.
